Excel Support

The support for convert records from/to Excel to plain records or only read or write excel worksheets is very limited and only works for simple scenarios

What version of the ExcelStorage must I use ?

ExcelNPOIStorage

Recomended. Uses the excellent library NPOI, there is no need to have Excel installed and has a great performance
